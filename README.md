# 🍊 Citrus Organellar Genome
![github](https://img.shields.io/badge/Citrus-orange)  ![github](https://img.shields.io/badge/Oranellar-pink)  ![github](https://img.shields.io/badge/Genetic-yellow)

We have released 82 non-redundant chloroplast genomes from natural citrus germplasm and 202 organelle genome drafts from artificial citrus hybrid populations. These findings will accelerate research on citrus organelle genomes, including but not limited to organelle genome editing, organelle genetics, and nucleoplasmic evolutionary conflicts.

Please cite：
>Sun X F, Wang X H, Yue H L, et al. Cytoplasmic inheritance analysis of citrus hybrid populations. Sci Sin Vitae, 2026, 56: 657–670, doi: 10.1360/SSV-2025-0276
